Preparing full name

Implement a function for outputting the full name of a person. The function takes two string arguments: firstName and lastName. You need to concat them together with a space in between.

If an argument is null, do not add it to the full name and do not insert a space.


Sample Input:
Joseph
Brown

Sample Output:
Joseph Brown


Sample Input:
Joseph
null

Sample Output:
Joseph



Memory limit: 256 MB
Time limit: 5 seconds
