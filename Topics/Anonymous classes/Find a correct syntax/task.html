Find a correct syntax

Here's an interface:

interface AnInterface {

    void method();
}

Select the correct syntax for creating an anonymous class.

a)

new AnInterface {

    @Override
    public void method() {
        // do something
    }
};

b)

AnInterface() {

    @Override
    public void method() {
        // do something
    }
};

c)

new AnInterface() {

    @Override
    public void method() {
        // do something
    }
};

d)

new AnInterface() {{

    @Override
    public void method() {
        // do something
    }
}};
