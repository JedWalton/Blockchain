Synchronize the code

Here's a class named SimpleClass. It has two static fields of different types and a method to initialize these fields.

public class SimpleClass {

    private static String strField;
    private static Integer numField;

    public static void initStaticFields(String str, Integer number) {
        strField = str;    // 1
        numField = number; // 2
    }
}

Multiple threads are trying to invoke initStaticFields method of the class. You need to synchronize this code so that a thread would not be able to set new values until another thread has not set both of them (in a way atomic operation does).

How can one do it? Select all correct cases.
