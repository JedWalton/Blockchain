Production of all numbers in the range

Write a lambda expression that accepts two long arguments as a range's borders and calculates (returns) the product of all numbers in this range (inclusively). It's guaranteed that 0 <= left border <= right border. If left border == right border, then the result is any border.

Solution format. Submit your lambda expression in any valid format with ; at the end.

Examples: (x, y) -> x + y; (x, y) -> { return x + y; };


Sample Input:
0 1

Sample Output:
0


Sample Input:
2 2

Sample Output:
2


Sample Input:
1 4

Sample Output:
24


Sample Input:
5 15

Sample Output:
54486432000



Memory limit: 256 MB
Time limit: 5 seconds
