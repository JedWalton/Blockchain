Log Search Facade

The search method of LogSearchFacade is intended to:

  • collect logs from standard input;
  • filter collected logs. Filtered ones should contain a passed expression;
  • format filtered logs.

There are 3 classes that help to achieve the goal: LogCollector, LogFilterand LogFormatter. Use them to complete the search method.


Sample Input:
6
INFO: Entity [id = 3] is created
ERROR: Failed to update [id = 3], no permissions
INFO: Security configuration is updated
INFO: Entity [id = 3] is updated
INFO: Entity [id = 3] is removed
ERROR: Entity [id = 3] doesn't exist

Sample Output:
Failed to update [id = 3], no permissions
Entity [id = 3] doesn't exist



Memory limit: 256 MB
